## Tech Stack

- **Framework**: Next.js 16
- **Styling**: Tailwind CSS v4
- **Typography**: Anton (headlines) + DM Sans (body)
- **Icons**: Lucide React
- **Deployment**: GitHub Pages / Vercel

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```
